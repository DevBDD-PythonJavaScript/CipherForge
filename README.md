# CipherForge 🔐

A lightweight encryption tool designed for secure communication between two trusted parties. One numeric seed generates one unique cipher.

## How It Works

1. Enter a numeric seed (e.g., `8374920158`)  
2. Share this number with your trusted contact  
3. Identical seeds produce identical ciphers — enabling seamless encryption and decryption

## Features

- Deterministic seed-based cipher generation  
- Full Unicode support (Cyrillic, Latin, digits, special characters)  
- Dark/light theme toggle (`Alt+D`)  
- Copy to clipboard functionality  
- Seed export/import for easy sharing
